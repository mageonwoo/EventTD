# EventTD
이벤트용 미니 타워디펜스 프로젝트입니다.

한 번의 플레이가 시작부터 종료까지 끊기지 않도록,\
작은 범위의 규칙과 제한된 기능만으로 구성했습니다.

## 프로젝트 개요
- 이벤트 콘텐츠를 가정한 소규모 미니게임
- 단일 세션 플레이 구조
- 정상 / 실패 / 종료 흐름을 포함한 플레이 완결
  
## 주요 포인트
- 시작부터 종료까지 이어지는 플레이 흐름
- 실패 및 종료 상황에서도 정상적인 마무리
- 입력 예외 및 리소스 조건에 따른 행동 제한
- 세션 종료 후 로비 복귀

## 디렉토리 구조
02.Scripts\
├─ Core        // 게임 흐름 및 주요 로직 및 설정 데이터\
├─ Runtime     // 플레이 관련 컨트롤\
└─ UI          // UI 구성
